# 优化后台管理，拆分路由

## 目标
将后台管理从单页 Tab 切换改为独立路由页面

## 路由结构
```
app/admin/
├── layout.tsx          # 共享布局（Header + Tab 导航）
├── page.tsx            # 重定向到 /admin/links
├── links/
│   └── page.tsx        # 链接管理
└── categories/
    └── page.tsx        # 分类管理
```

## 执行步骤
1. 创建 app/admin/layout.tsx - 共享布局
2. 修改 app/admin/page.tsx - 重定向
3. 创建 app/admin/links/page.tsx - 链接管理
4. 创建 app/admin/categories/page.tsx - 分类管理
5. 清理旧组件
